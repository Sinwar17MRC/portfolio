##  Key Features

- **Responsive Design**: Optimized for everything from 4K monitors to small mobile screens with safe horizontal clipping.
- **Dynamic Arsenal**: Categorized skills grid featuring advanced AI, BI, and MLOps tools.
- **High-Performance Carousel**: Interactive projects showcase with construction state management and custom scroll tracking.
- **Absolute Coordinate Navigation**: Precision smooth-scrolling that accounts for header offsets and reveal-wrappers.
- **Automated Contact**: Pre-configured `mailto` integration and high-visibility social bubbles.
- **Neural Aesthetic**: Custom-engineered background effects including a neural-network SVG and horizontal ticker ribbons.
